# Tic-tac-toe

  This is a tic-tac-toe console game written in Ruby.
  There are 2 human players and a board with the number of raws (and columns) defined by a player and limited from 3 to 9 rows. 

## General 

1. A game continues up to 3 rounds. 
2. Each step is followed by instructions.

## Rounds 

1. Each round lasts until a player wins or until all the cells are occupied by bids. After the rounds are over, a game winner is announced. 
2. At any round time when making a bid a player can stop a game, skip the round and start a new round (if there are any left out of three rounds) or start a new game by entering stop-words accordingly. 
3. If any sequential columns, rows or diagonal rows are occupied with 'x's or 'o' accordingly, then a round winner gets a score.

## Players

1. The first player gets 'x' and the second player gets 'o' by default. 
2. Each player enters a name or nothing, if nothing, then a player is named 'Stranger'. 
3. Each player gets their turn to enter their bid. 


## Board and player bids 
1. There are letters and numbers at the left and at the top of the board accordingly to make choice easier. 
2. A user enters a letter and a number to make a bid. 
3. Each time a board displays current state of the board based on values entered. 
4. A player may not make a bid at the cell that does not exist or already has a value.
