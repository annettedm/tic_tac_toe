tic tac toe game in a console

This is a tic-tac-toe console game written in Ruby.
There are 2 human players and a board with the number of raws (and columns) defined by a player and limited from 3 to 9 rows. The first player gets 'x' and the second player gets 'o' by default. Each player enters a name or nothing, if nothing, then a player is named 'Stranger'. 
Each player gets their turn to enter their bid. There are letters and numbers at the left and at the top of the board accordingly to make choice easier. A user enters a letter and a number to make a bid. Each time a board displays current state of the board based on values entered. A player may not make a bid at the cell that does not exist or already has a value. If any sequential columns, rows or diagonal rows are occupied with 'x's or 'o' accordingly, then a round winner gets a score. A game continues up to 3 rounds. Each round lasts until a player wins or untill all the cells are occupied by bids. After the rounds are over, a game winner is announced. 
At any round time when making a bid a player can stop a game, skip the round and start a new round or start a new game by entering stop-words accordingly. 
Each operation is followed by instructions.
